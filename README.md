# Baccharis OS - Sistema Central

Bem-vindo ao repositório do Gêmeo Digital da Baccharis Bio Tech. 
Este sistema integra os processos da nossa biorrefinaria com monitoramento em tempo real.

## Funcionalidades
- Interface de controle da Biorrefinaria.
- Monitoramento de sensores de hardware.
- Integração para suporte à agricultura familiar.

## Tecnologias
- Desenvolvido em Python com o framework Streamlit.
